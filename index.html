Cloud Computing

Basiswissen

Was ist Cloud Computing?

Cloud Computing stellt über das Internet Ressourcen zur Verfügung, für die man nur so lange zahlen muss, wie man sie benutzt. Im Gegensatz zur Cloud speichert man dort keine Daten, sondern Applikationen oder Rechenleistung.

Was ist eine Cloud?

Eine Cloud ist ein Netzwerk aus Servern, die über das Internet miteinander verbunden sind. Die Server können sich überall auf der Welt befinden. Die Cloud ist also ein Netzwerk aus Servern, die über das Internet miteinander verbunden sind.

Was ist IaaS?

  • Infrastructure as a Service
  • IaaS stellt eine Cloud-Infrastruktur bereit, die es Benutzern ermöglicht, virtuelle Server, Netzwerkressourcen und Speicherplatz zu mieten.
  • Die Benutzer verwalten und kontrollieren die virtuellen Maschinen und die darauf ausgeführten Anwendungen.
  • Beispiele für IaaS-Anbieter sind Amazon Web Services (AWS), Microsoft Azure und Google Cloud Platform.

Was ist PaaS?

  • Platform as a Service
  • PaaS ist eine Cloud-Plattform, die es Benutzern ermöglicht, Anwendungen zu entwickeln, auszuführen und zu verwalten, ohne sich um die zugrundeliegende Infrastruktur kümmern zu müssen.
  • Die Benutzer können auf einer Plattform programmieren, die bereits alle notwendigen Tools und Ressourcen enthält.
  • Beispiele für PaaS-Anbieter sind Heroku, Google App Engine und Microsoft Azure.

Was ist SaaS?

  • Software as a Service
  • SaaS ist ein Cloud-Service, bei dem Anwendungen vollständig in der Cloud gehostet werden und über das Internet zugänglich sind.
  • Die Benutzer mieten die Anwendungen, anstatt sie auf ihren eigenen Geräten zu installieren.
  • Beispiele für SaaS-Anbieter sind Salesforce, Dropbox und Google Workspace.

Was ist Docker?

Docker ist eine Open-Source-Software, die es ermöglicht, Anwendungen in sogenannten Containern zu verpacken. Container sind eine Art virtuelle Maschinen, die jedoch nicht das gesamte Betriebssystem enthalten, sondern nur die Anwendung und alle Abhängigkeiten, die sie benötigt, um zu funktionieren.
